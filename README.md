# Data Structures From Scratch (C++)

## Goal
Implement core data structures from scratch using C++.

## Implemented Data Structures
1. Singly Linked List
2. Binary Search Tree (BST)

## How to Run
Compile using:
g++ linked_list.cpp -o linkedlist
g++ binary_search_tree.cpp -o bst

Run using:
./linkedlist
./bst

## Explanation
Each data structure is implemented from scratch without STL containers.
Insert, delete, and search operations are demonstrated in main().
Time complexity is mentioned in comments.

This project was tested using an online C++ compiler with code and output screenshots included in the documentation files.

## Time Complexity
Linked List:
- Insert: O(n)
- Delete: O(n)
- Search: O(n)

Binary Search Tree:
- Insert/Search/Delete: O(h) where h is the height of the tree
